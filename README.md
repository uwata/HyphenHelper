# Photoshop plugin using a local library to show you where hyphens can go in a word.

- Useful for typesetters probably? I was just sick of alt tabbing to a slow website.
- To install just download and open com.wata.hyphenator_PS.ccx which should work if you own Photoshop so you might have to look for how to manually install ccx plugins otherwise
- Works for Photoshop version 22.0 and up
- BTW YOU DON'T HAVE TO PRESS THE BUTTON YOU CAN ALSO JUST PRESS ENTER TO SUBMIT
